# **Guia de Markdown para GitHub**

- [x] Negrito `** ou __`
- [x] Italico `* ou _`
- [x] Riscado `~~`
- [x] Titulo `#`
   * Outros Titulos `##` `###` ...
- [x] Linha `*** ou ---`
- [x] Emoji `:`
- [x] Quote `>`
---

## __*Para Listas*__ 
- [x] Lista Numerada `N+.`
- [x] Lista Demarcada `* ou -`
- [x] Lista de Tarefas `- [ ]/- [x]`
